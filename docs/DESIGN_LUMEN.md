# Дизайн-система Lumen (зафиксировано)

Документ **обязателен** для любой верстки и стилей приложения 22 QR (квест на **11** малых шагов после ввода). Визуал **не** менять и **не** «улучшать по вкусу» без **явной просьбы** владельца репозитория; отклонения — только после правки этого файла.

**Эталон для глаз:** статичный превью [`design-lab/index.html`](../design-lab/index.html) (те же атмосфера, стекло, иерархия). Реализация в React должна **визуально совпадать** с этим эталоном по токенам и принципам ниже.

**Связанные опоры:** [`PROJECT_CONCEPT.md`](./PROJECT_CONCEPT.md), [`PROJECT_SPEC.md`](./PROJECT_SPEC.md).

---

## 1. Имя и характер

**Lumen** — светлое утро, воздушное матовое стекло, спокойный лилово-персиковый фон без кислотности. Солидно и тепло, без детского «игрушечного» UI.

---

## 2. Типографика

| Роль | Шрифт | Примечание |
|------|--------|------------|
| Крупные заголовки экранов, акценты финала | **Cormorant Garamond** (500–600, при необходимости italic) | Не заменять на системный serif |
| Весь UI, кнопки, подписи, заглушки | **DM Sans** (400–600) | optical sizing включить в CSS где поддерживается |
| Редкий брендовый акцент (eyebrow / метки секций в лабе; в приложении — опционально мелкие лейблы) | **Syne** (600–700) | Только точечно, не для основного текста |

Подключение: Google Fonts, те же семейства, что в `design-lab/index.html`.

---

## 3. Цветовые токены (CSS-переменные)

Имена закреплены; значения **не** плавать без обновления этого документа:

| Токен | Значение | Назначение |
|-------|----------|------------|
| `--lum-ink` | `#1c1d26` | Основной текст |
| `--lum-muted` | `rgba(28, 29, 38, 0.56)` (или эквивалент) | Вторичный текст |
| `--lum-accent` | `#5b4bdb` | Акцент, активные сегменты, ссылки-кнопки |
| `--lum-accent-deep` | `#4532b8` | Низ градиента primary-кнопки |
| `--lum-accent-soft` | `rgba(91, 75, 219, 0.12)` | Пиллы, светлые заливки |
| `--lum-glass` | `rgba(255, 255, 255, 0.46)` | Заливка стеклянных карточек |
| `--lum-glass-border` | `rgba(255, 255, 255, 0.72)` | Обводка карточек |

Фон страницы — **мультислойный градиент**, как в эталоне (не один плоский цвет):

- Базовые оттенки: `#eef1ff` → `#f8f6ff` → `#fdfbff`
- Радиальные пятна: лилово-голубое (сверху слева), персиково-розовое (справа), при необходимости третье мягкое снизу

Поверх — **лёгкий noise** (SVG/opacity), `mix-blend-mode: overlay`, как в лабе.

---

## 4. Стекло (карточки)

- `backdrop-filter: blur(20–24px) saturate(140–160%)` (и webkit-префикс)
- Скругление карточек: **18–22px**
- Тень: мягкая, с лёгким **inset** сверху (светлая линия), без тяжёлого «неоморфизма»
- Граница: полупрозрачная белая, не серо-чёрная

---

## 5. Компоненты (поведение внешнего вида)

- **Primary CTA:** градиент `accent` → `accent-deep`, белый текст, **полная ширина** на мобильном, высота щедрая; тень фиолетовая мягкая.
- **Ghost / вторичная:** полупрозрачное белое стекло, тонкая светлая обводка (как «Хочу ещё раз прочитать позже» в лабе).
- **Пилл шага:** uppercase, letter-spacing, фон `--lum-accent-soft`, текст `--lum-accent`.
- **Прогресс:** 11 сегментов, заполненные — градиент accent; следующий — обводка / подсветка «next»; пустые — приглушённый серый.
- **Подарок:** слой 1 — Cormorant italic, спокойный; слой 2 — в рамке dashed + мягкий фон `accent-soft`; бейдж с иконкой подарка как в эталоне.
- **Заглушки:** те же карточки стекла, акцентная обводка слегка фиолетовая (`accent` с низкой opacity).

Финальный экран (титры): крупный Cormorant, центрирование, разделитель-линия градиентом accent.

---

## 6. Что запрещено без согласования владельца

- Смена палитры на другую (например тёмная тема, терракотовый «Aube», холодный «Meridian»)
- Замена шрифтов на другие семейства
- Отказ от стекла в пользу плоских сплошных карточек без `backdrop-filter`
- «Упрощение» ради скорости, которое ломает токены выше
