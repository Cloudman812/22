# Локальный превью без интернета (опционально)

Основной способ тестов и «дня Х» — **HTTPS в интернете**, см. [`DEPLOYMENT.md`](./DEPLOYMENT.md) (GitHub Pages).

Этот файл — краткая памятка, если нужно поднять **только в квартире** раздачу папки `dist` или лаборатории дизайна по Wi‑Fi.

## Быстро

После `npm run build` удобнее превью Vite (учитывает `base: /22/`):

```bash
npm run preview:lan
```

На телефоне: **`https://<IP-Mac>:4173/22/`** (HTTPS из превью; при предупреждении о сертификате — «Продолжить»).

Камера на iPhone по **http + IP** в Safari часто **не** работает; для проверки сканера используй задеплоенный **HTTPS** URL.

## Дизайн-лаборатория

```bash
npm run serve:lab
```

То же порт **8080**, см. скрипты в `package.json`.
